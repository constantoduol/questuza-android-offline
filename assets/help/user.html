Creating a User

To create a user :
  1. Enter the user's email address
  2. Enter the user's name
  3. Select the role the user plays such as 'Seller' or 'Admin'
  4. Click on the 'Create User' button

Updating a user

To update a user:
  1. Search for the user under 'Search Users'
  2. Select the user you want to update from the drop down by clicking on the email address
  3. Edit any values that you want to update
  4. Click on the 'Update User' button

Enabling, disabling, deleting or reseting password for user account

  1. Search for the user under 'Search Users'
  2. Select the user you want to enable, disable ... etc from the drop down by clicking on the email address
  3. Click on the appropriate button to complete the action

Viewing all users

To view all users created under your business click on the blue search button
